 TryHackMe – Pre Security Path Notes

 What This Path Covered
The Pre Security path is the foundation for everything in cybersecurity. It's split into
networking, web fundamentals, and basic security/IT concepts.

 Networking Basics
- **OSI Model**: 7 layers (Physical, Data Link, Network, Transport, Session, Presentation,
  Application) — a mental model for how data moves across a network.
- **TCP/IP Model**: The real-world model used on the internet (4 layers). TCP is
  connection-based and reliable; UDP is faster but doesn't guarantee delivery.
- **IP Addressing**: IPv4 vs IPv6, public vs private IPs, subnetting basics.
- **Ports & Protocols**: Common ports to know — 80/443 (HTTP/HTTPS), 22 (SSH), 21 (FTP),
  25 (SMTP), 53 (DNS), 3389 (RDP).
- **DNS**: Translates domain names into IP addresses. Understanding DNS is key for later
  spotting things like DNS tunneling or malicious domains.

 Web Fundamentals
- How a browser request actually works: DNS lookup → TCP handshake → HTTP request → server
  response.
- HTTP vs HTTPS and why encryption (TLS/SSL) matters.
- Basic structure of client-server communication.

 Core IT/Security Concepts
- Operating system basics (Windows & Linux file systems, permissions).
- Introduction to the CIA Triad: Confidentiality, Integrity, Availability.
- Basic terminology: vulnerability, exploit, threat, risk.
